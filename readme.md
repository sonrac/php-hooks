# Pre commit hook

## Enable hook 

```shell
make enable-pre-commit-hook
```

## Disable hook 

```shell
make disable-pre-commit-hook
```

## Disable hook for commit

Uncheck checkbox in commit modal dialogue 
if you need commit without pre-commit hook checks
